# Student Management System

This Python project is designed for student management. _The program is modular and allows users to add, search, update, and delete students_, as well as list them so that all registered students can be viewed, along with information about the course they are enrolled in and its status.

## Features

| option | Descripción |
|--------|-------------|
|1. Register student|_Add new students_|
|2. List students|_Shows all registered students_|
|3. Search student|_Search for a specific student_|
|4. Update student|_Update a specific student_|
|5. Remove student|_Remove a specific student_|
|6. Exit|_Close the program_|

## How the program works

When you start the program, a menu will be displayed so you can see the available options.

**1. Register student**

- This option allows you to add a new student and register them in a course (upon registration, the status will automatically change to active). You must enter the student's name, age, and select the course. Once you complete these steps, the system will generate a unique ID.

**3. List students**

- This option displays all students registered in the system, their personal information (ID, name, age), the course they are enrolled in, and its status.

**4. Search student**

- This option allows you to search for a student by ID or name and displays the information for that specific student.

**5. Update student**

- This function allows you to update the course a student is enrolled in and change their status.

**6. Remove student**

- This function allows you to remove a specific student.

**7. Exit**

- This option closes the program.
